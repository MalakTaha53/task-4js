Fetch

not need to install its build in js

Syntex :

const response = await fetch(`https://forkify-api.herokuapp.com/api/search?q=pizza`,{method:'GET'}); or

const response = await fetch(`https://forkify-api.herokuapp.com/api/search?q=pizza`,{method:'POST'});

if not use await will print (Promise {pending}) so we use await to get the response also the function type is async

the response contain information about request not data that we needed so to get data from the response need to using json() method with await becouse json take time to get data

Axios

we need to install it using cdn or npm

Syntex:

const response = await axios.get('https://dummyjson.com/products'); or

const response = await axios.post('https://dummyjson.com/products'); or

const response = await axios.delete('https://dummyjson.com/products');

if not use await will print (Promise) so we use await to get the response also the function type is async

the response return data direct without using json() method
